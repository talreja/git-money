仕事の質が全て

住んでる国なんて関係ないのです。アメリカだろうが、ウクライナだろうが、島だろうが、インドだろうが。もしコーディングの問題解決が出来たのであれば、それに見合った支払いがされるべきです。git money　のフォーカスはただ一つ。仕事の質、それだけです。

コーディング出来ない？大丈夫です！

存在するのはテクニカルな問題だけではありません。他にもグラフィック、テキスト、ビデオ編集、ソーシャルメディアのアシスタントなど、色々あります。もちろん、お金を貰えます！どんなスキルであっても、ここで生かせればお金が稼げるのです。

使い方

ビットコインでの支払い可能なプロジェクトを見付けて、国境を超えた取引を楽しみましょう

git チャンス

現時点で上がっているビットコインでの支払い可能な問題を、GitHub上で探してみましょう。 緑のレベルが付いている問題を選ぶと、そこに支払い金額の表示されます。現時点での支払い方法は、ビットコイン、または米ドルのみです。問題について不明な点があれば、いつでも気軽に質問して下さいね！ 初めての方はこちら

git ワーク

仕事のチャンスを見付けたら、早速やってみましょう！レポをクローンし、ブランチを作成し、変更内容をコミットして、プルリクエストを提出するだけです。プルリクエストの中に、 問題解決が終了した、とメモを添えて下さい。 ビットコインのアドレスもお忘れなく。　　　　　　　　　　ビットコイン初めての方はこちら

git マニー

プルリクエストがマージされると、その時点で即座に支払いが受け取れます。どの様に git money を使おうと、あなた次第です。小さな問題解決を複数してそれなりに稼ぐ人もいれば、最初から大きな仕事を好んでする人もいます。もしかしたら、あなたはただお小遣いを稼ぎたいだけかも知りません。いずれにせよ、あなた次第です。
