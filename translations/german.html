Wie es funktioniert

Finden Sie eine Bitcoin-Projekt zu zahlen und randlos Handel genießen.

git Chancen

Stöbern aktuellen Bitcoin-zahlbaren Ausgaben auf GitHub . Alle Probleme mit ein Auszahlung wird das grüne Label git Geld zusammen mit einem "Aktuelle enthalten Bounty:" wenn Ansichten. Die aktuelle Auszahlung für die Behebung dieses Problems ist bezeichnet in BTC (Bitcoin) und US-Dollar. Wenn Sie eine Frage zu die Ausgabe, fühlen sich frei, sich dazu zu äußern und fragen Sie an! Neu bei git?

git arbeit

Sobald Sie eine Gelegenheit gefunden haben, sind Sie auf halbem Weg nach Hause (oder wo auch immer es ist, du gehst!). Gehen Sie voran und an die Arbeit! Alles, was Sie tun müssen, ist Klon das Repo, erstellen Sie einen Zweig, Ihre Änderungen und einen Pull einreichen anfordern. In Ihrem Pull-Request-Nachricht, stellen Sie sicher, zu beachten, dass es das Problem behebt und auch sind Ihre Bitcoin-Adresse. Neu bei Bitcoin?

git Geld

Andere gehen gerade für die großen Jobs. Möglicherweise sind Sie Mitte der Straße, Kommissionierung und der Wahl zusätzliches Geld auf das zu machen, Seite.
